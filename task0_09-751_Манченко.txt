Тема 17. Электронный кинотеатр.
1. Бизнес-требования 
Электронный кинотеатр обеспечивает возможности: 
-посетителю кинотеатра - приобрести билеты на интересующий его сеанс 
-посетителю кинотеатра - подобрать удобное для него время для похода на киносеанс 
-посетителю кинотеатра - прочитать информацию об интересующем его фильме 
-кинотеатру - использовать дополнительный канал продаж билетов 
2. Бизнес-правила 
Электронный кинотеатр должен следовать таким законам, как
 - Федеральный закон №152-ФЗ (ред. от 31.12.2017) "О персональных данных";
 - Закон РФ №2300-1 (ред. от 18.03.2019) "О защите прав потребителей".
3. Требования пользователей 
Пошаговый сценарий покупки интересующего билета. 
Набор uses stories для покупки билетов в электронном кинотеатре: 
-для сравнения фильмов; 
-для добавления билетов в корзину; 
-для управления корзиной; 
-для оформления заказа (ввод данных пользователя, ФИО, e-mail); 
-для онлайн-оплаты. 
4. Атрибуты качества 
1) Электронный кинотеатр должен быть прост и понятен в использовании даже непродвинотому ПК-пользователю. 
2) Электронный кинотеатр жолжен быть способен к взаимодействию с пользователем. Например, при неверно введенных данных выводить окно ошибки. 
3) Электронный кинотеатр должен иметь привлекательный внешний вид. 
5. Ограничения 
1) При покупки онлайн-билетов пользователю необходимо после ввода данных карты ввести код-пароль, который придет по номеру телефона. 
2) Проверять, остались ли на выбранное время/место свободные билеты. 
6. Внешние интерфейсы 
1) Взаиимодействие с платежной системой при покупке онлайн-билетов. 
2) API социальных сетей для оповещения пользователей о новинках кинотеатра. 
3) Взаимодействие с сервером электронного кинотеатра для резервирования и оплаты билетов. 
7. Системные требования 
Доступ к операциям со счетами пользователей должен обеспечиваться через интернет-банк пользователя в новом всплывающем окне. 
8. Функциональные требования 
1) Необходимо реализовать строку поиска. 
2) Сформировать сортировку по новизне добавляемых фильмов/умолчанию.
